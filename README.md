## Getting Started
1. git clone https://github.com/JoaoLSA/VisualStudioSnippets.git
2. Add snippet to Visual Studio

	Ctrl + K, Ctrl + B
Then click add and select VisualStudioSnippets folder and ok

3. Its done!
